### Notas:
### Para correr la api rest de Django:
* Ubicarse en ...back\in47\library_site

Ejecutar:
* python manage.py migrate
* python manage.py runserver

### Para correr el proyecto de Angular
* Ubicarse en ...\front\library-in47

Ejecutar:
* ng serve -o
